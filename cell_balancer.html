Cell Balancer

The Maccor Cell Balancer is a cutting-edge device designed to charge and discharge individual cells in a string (battery pack) simultaneously. Each cell balancer can handle up to 8 cells, and by using multiple balancers in series, larger battery packs can be balanced effectively. Maccor's innovative 'Mactest32.exe' software includes integrated cell balancing features, empowering users to incorporate cell balancing steps seamlessly into their test procedures.

Specification Value
Model 1205A
Cells per Box 8
Maximum Current per cell 1000 mA
Cell Connection 2-wire
Series Configuration Yes
Stand-Alone Mode Yes
