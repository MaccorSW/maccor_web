Advanced Multiple Level Pulsing (AMLP)

Powerful precision for real-world pulse testing—now with visual proof.

Modern electronics demand smarter battery testing—and Maccor delivers.

Our new Advanced Multiple Level Pulsing (AMLP) feature empowers you to simulate real-world dynamic load conditions with unmatched precision and control.

Features:

• Modes:Supports constant current, constant power, and constant resistance (discharge only).

• Duration: Programmable from 100 μs to 15 seconds with 1-μs increments.

• Samples: Report intervals from 100 μs to 15 seconds with 1-μs increments.

• Levels: Up to 15,000 levels possible.

• Profiles: Multiple AMLP profiles can be included in a single test procedure.

• End Conditions: up to two voltage end conditions.


Whether you're evaluating mobile devices, IoT tech, or EV subcomponents, AMLP makes it easy to replicate GSM, LTE, and other dynamic loads with confidence.

An Advanced Multiple Level Pulse is implemented as a Step Type within a test procedure. The first step in configuring AMLP is to create a pulse profile (*.amlp file).


Maccor provides the MacAMLP.exe application for this purpose.

Application Examples

AMLP profiles are versatile and can simulate a variety of real-world applications such as:

• Mobile communications (e.g., GSM, LTE load patterns)

• Wearable electronics and IoT devices

• High-speed digital processing loads

• Electric vehicle subcomponents


Maccor’s multi-level pulse editor is a powerful tool designed to simplify the creation of high-speed pulse trains. Because pulse train definitions are stored in an ASCII tab-delimited format, they can be created or modified using any application capable of saving plain text files—offering maximum flexibility in pulse profile development.
