Ultra Speed Battery Tester 

Upgrade to Maccor's newest test platform, The Series 6000. While maintaining all of the features of Maccor's high precision hardware, the Series 6000 adds an element that will elevate your testing. You will be amazed by the new high-speed control and data collection.
The Advanced Multiple Level Pulse ("AMLP") allows you to create the most complex pulse profiles available.

Advanced Multiple Level Pulse

AMLP

  • Step can consist of a set of up to 15,000 levels
  • Each level can be constant current, power, or resistance
  • Each level will be a minimum of 100 microseconds and a maximum of 15 seconds
  • Levels can be programmed in 100-microsecond increments
  • All levels in a particular AMLP step must be in charge, or all must be in discharge
  • Voltage end condition checked every 100 microseconds
  • When the voltage end condition is met the test will branch to a user selected step elsewhere in the test procedure
  • Multiple AMLP steps can be programmed in a given test procedure as long as the sum does not exceed the 15,000 level limit and 225 second limit
  • Record data as fast as every 100 microseconds

 

Additional Features

Data Collection

Program test steps that log data on multiple factors like current, voltage, time, and more! Collect data continuously at a speedy 1 kHz rate from various test channels. The first milliseconds of each data point showcase voltage and current readings at an exhilarating 20 kHz rate.

Step Length and Control

Specify test steps as short as 750 microseconds and increase them in one microsecond increments!

Set Point Lock-In Time

Lock In Setpoint in as little as 1 millisecond.

Waveforms

Stream external text test files using waveform steps, with each level customizable as constant current or power, programmable in increments as short as 10 milliseconds with a time resolution of 1 microsecond. There is no limit to the number of levels in a waveform, and multiple unique waveforms can be run in the same test.
